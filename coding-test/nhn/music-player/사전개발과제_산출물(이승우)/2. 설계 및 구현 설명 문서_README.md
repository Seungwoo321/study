# 🚀 Welcome


## 프로젝트 시작
* 빌드 하고 루트에 위치한 index.html 을 실행 
``` 
npm run build
```


## 설계 및 구현에 대한 설명
 * 크게 재생부와 목록 리스트로 나누고, 각 기능을 *Factory.js 에서 정의
 * 기능들이 UI 이벤트 및 사용자에 의해 실행되는 것을 Observer 에서 등록 하여 감시하고 통신함
 * render.js 에서 실제 돔을 조작하고 렌더링

## __재생부__

### 속성
- 타이틀
- 남은 재생 시간 

### 기능 
- 타이틀 디스플레이
- 남은 시간 리얼타임 갱신
- 진행 시간 프로그레스바 
- 이전 곡 버튼
- 일시정지 버튼
- 멈춤 버튼
- 다음 곡 버튼

## __목록 탭__

### 속성
- 목록 이름
- 목록의 상태 (재생중인지, 선택되었는지)

### 기능
- 목록 삭제 
- 목록 선택 


## __음원목록 && 재생목록__

### 공통 속성
- 타이틀
- 총 플레이 시간 

### 공통 기능 
- 목록에서 선택 
- 목록에서 삭제 

### 음원 목록 기능
- 재생목록 생성 

### 재생 목록 기능 
- 가나다 정렬
- 랜덤 정렬
